# WifiForge Road Map

## v3.x.x

- WPS Lab - Currently the WPS Pixie Dust Attack lab does not work, this is under investigation.

## v4.0.0

Currently v4.0.0 is under development with the hopes to implement the following features:

- WifiProbe - A snapshot tool that take a snapshot of the wireless landscape around the host, making a 1:1 copy of the Access Points.
- WifiHound - A output file type for WifiProbe that formats an output file that can be fed into BloodHound to check different analytics and Access Point information.
- Drone Hacking - Mininet-Wifi has the capability to do wireless drone hacking labs, we plan to impliment this into WifiForge.
